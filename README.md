React-приложение, созданное с помощью Vite. Демонстрирует компонентный подход с использованием Bootstrap для стилизации.

## Технологии

- **React 18** - библиотека для построения пользовательских интерфейсов;
- **Vite** - быстрый сборщик и инструмент разработки;
- **Bootstrap 5** - CSS фреймворк для стилизации;
- **JavaScript (JSX)** - синтаксис для React компонентов.
